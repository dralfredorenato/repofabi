# repofabineuro

Sistema da Clínica Dra. Fabiana Mugnol usando Next.js 14 + TypeScript + Tailwind + Supabase.

## Rodar localmente

```bash
npm install
npm run dev
```

## Entregue

### Sprint 1
- Estrutura base de rotas (clínica/paciente/auth)
- Layout inicial da clínica
- Middleware para rotas protegidas
- Helpers Supabase (browser/server)

### Sprint 2
- Login real com Supabase Auth
- Logout no topbar da clínica
- CRUD de demandas (criar, listar, filtrar, atualizar status, detalhe)
- Dashboard adaptativo por perfil
