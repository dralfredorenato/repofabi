# repofabineuro

Fundação inicial do sistema da Clínica Dra. Fabiana Mugnol usando Next.js 14 + TypeScript + Tailwind.

## Rodar localmente

```bash
npm install
npm run dev
```

## Entregue

- Estrutura base de rotas (clínica/paciente/auth)
- Layout inicial da clínica
- Middleware para rotas protegidas
- Helpers Supabase (browser/server)
